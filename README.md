# Overview

Crowdsourced list sorting.

# Setup

## Requirements

* Python 3.6+

## Installation

Clone the source code:

```sh
$ git clone https://github.com/jacebrowning/crowdsorter.git
$ cd crowdsorter
```

Install project dependencies:

```sh
$ make install
```

# Usage

Run the application:

```sh
$ make run
```
